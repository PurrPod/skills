# 章节名称 - 自测题

## 一、选择题

1. 在计算机网络中，负责在不同网络之间路由数据包的层次是哪一层？
- [ ] A. 物理层
- [ ] B. 数据链路层
- [x] C. 网络层
- [ ] D. 传输层
> 解析：网络层负责逻辑寻址和路由选择，如 IP 协议。

2. 根据广义相对论，以下关于引力波的描述哪项是正确的？
- [ ] A. 引力波是电磁波的一种
- [x] B. 引力波是时空曲率的涟漪，以光速传播
- [ ] C. 只有带电荷的物体加速运动才会产生引力波
- [ ] D. 引力波可以通过法拉第笼屏蔽
> 解析：引力波由质量的加速度产生，是时空本身的波动（涟漪），真空中传播速度恒为光速 $c$。

## 二、判断题

- [ ] 1. 系统综合研究系统对于输入激励信号所产生的响应。
- [x] 2. 能量信号的平均功率恒为0。
> 解析：第1题描述的是系统分析（Analysis），而不是系统综合（Synthesis）。

## 三、填空题

1. 在 Linux 中，用于更改文件权限的命令是 ____。
答案：chmod
> 解析：chmod (change mode) 用于修改文件权限。

2. 理想气体状态方程的数学表达式为 ____。
答案：pV = nRT
> 解析：其中 $p$ 为压强，$V$ 为体积，$n$ 为物质的量，$R$ 为理想气体常数，$T$ 为热力学温度。

## 四、计算与大题

1. 已知 LTI 系统的微分方程为 $y''(t)+3y'(t)+2y(t)=x(t)$，求系统的单位冲激响应 $h(t)$。
答案：
> 解析：
> **解题步骤：**
> (1) 写出特征方程：$s^2+3s+2=0$
> (2) 求解特征根得：$s_1=-1,\;s_2=-2$
> (3) 设系统齐次解形式为 $h(t)=(Ae^{-t}+Be^{-2t})u(t)$
> (4) 根据初始条件 $h(0^+)=0, h'(0^+)=1$ 代入解得：$A=1, B=-1$
> (5) 最终结果：$h(t)=(e^{-t}-e^{-2t})u(t)$
> **边界检验：** 当 $t \to \infty$ 时，$h(t) \to 0$，符合稳定系统的特征。

2. (算法题) 请用 Python 写出二分查找的代码骨架。
答案：
> 解析：
> 核心思路是维护左右指针，每次折半。
> ```python
> def binary_search(arr, target):
>     left, right = 0, len(arr) - 1
>     while left <= right:
>         mid = (left + right) // 2
>         if arr[mid] == target:
>             return mid
>         elif arr[mid] < target:
>             left = mid + 1
>         else:
>             right = mid - 1
>     return -1
> ```
> **极简走查：** 假设 `arr=[1,3,5]`, 找 `5`。初始 `left=0, right=2, mid=1(指向3)`。因为 `3 < 5`，下一步 `left=2`，此时 `mid=2` 指向 5，命中返回。
